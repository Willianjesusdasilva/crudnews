# CrudNews

## Run application.

This application runs on docker-compose, to execute use the command below in the project's root folder.
```
$ docker-compose up -d
```

## Run tests.
To run the tests, use lib pytest with the command inside the  folder ./tests.
```
pytest test.py
```

## Libs.
-Flask
-Pytest
-Pymongo>=2.7.1
-Mongoengine
